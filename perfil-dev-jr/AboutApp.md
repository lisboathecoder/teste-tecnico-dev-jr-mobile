# Perfil do Desenvolvedor Júnior
### Qual o objetivo do app
> Aplicativo criado em React Native com Expo para apresentar um perfil profissional.
---
### Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- React Navigation
- Git
- GitHub

---
### Funcionalidades
- Exibe dados do candidato
- Apresenta uma tela de perfil
- Lista habilidades
- Mostra projetos
- Exibe informações de contato
- Possui navegação por abas inferiores

## Como executar o projeto
1º passo - Clonar este repositório
```bash
git clone https://github.com/lisboathecoder/teste-tecnico-dev-jr-mobile
```
2º passo - Entrar na pasta e instalar as dependências
```bash
cd teste-tecnico-dev-jr-mobile
cd perfil-dev-jr
npm install
```
3º passo - Após fazer suas mudanças, rode o aplicativo!
```bash
npm start
```
